# Basic Vehicle

This folder contains a minimal differential-drive vehicle for Gazebo Sim and a ROS 2 keyboard teleop script.

## Files

- `models/basic_vehicle/model.sdf`: four driven wheels in a skid-steer layout, with a Gazebo diff-drive system that rotates all wheel joints from velocity commands
- `worlds/basic_vehicle_world.sdf`: flat ground plane with the vehicle spawned at the origin
- `drive_vehicle.py`: ROS 2 keyboard teleop that publishes `geometry_msgs/msg/Twist`

## Run

From the repository root:

```bash
export GZ_SIM_RESOURCE_PATH=$PWD/ros2-experiments/basic-vehicle/models:$GZ_SIM_RESOURCE_PATH
gz sim ros2-experiments/basic-vehicle/worlds/basic_vehicle_world.sdf
```

In a second terminal, bridge ROS 2 `Twist` to Gazebo:

```bash
ros2 run ros_gz_bridge parameter_bridge /cmd_vel@geometry_msgs/msg/Twist@gz.msgs.Twist
```

In a third terminal, start keyboard control:

```bash
python3 ros2-experiments/basic-vehicle/drive_vehicle.py
```

## Controls

- `w`: faster forward
- `s`: faster reverse
- `a`: turn left faster
- `d`: turn right faster
- `x` or `space`: stop
- `q`: quit

## Notes

- The Gazebo model uses `gz::sim::systems::DiffDrive`, so wheel joint rotation is generated by the simulator rather than faked in the teleop script.
- The front and rear wheel pairs on each side are both driven, so the vehicle behaves like a simple 4-wheel skid-steer robot.
- If your Gazebo installation uses older `ignition-*` plugin filenames, update the plugin `filename` values in the SDF accordingly.
